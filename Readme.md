## 🛠️ Como Usar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/prl.git
cd prl
```

2. Instale as dependências:

```bash
npm install
```

3. Execute a aplicação:

```bash
npm run dev
```

O PRL estará disponível em `http://localhost:3000`.
